# 整数的扩展和溢出

## 整数的扩展

无符号扩展

    movzx ebx, al

有符号扩展

    movsx ebx, al; 0xffffffff

## 溢出

试图表示自身表示不了的数，就是溢出。

无符号数 --> CF

有符号数 --> OF

127 = 0b_0111_1111

0b1000_0000 = -128

-127 = 0 - 127 = 0b10000_0000 - 0b_0111_1111

= 0b_1000_0001 = -127 

-127 - 1 = 0b_1000_0000 = -128

$$x < 0 \Rightarrow x_b + |x| = 0 \Rightarrow x_b = 0 - |x|$$
